# Python Sales Analysis Project

## Project Overview

This project analyzes a sales dataset using Python.

## Tools Used

- Python
- NumPy
- Pandas
- Matplotlib
- Google Colab

## Project Tasks

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

## Business Insights

- North region generated the highest revenue.
- South region generated the lowest revenue.
- Online payment was the most preferred payment method.
- Laptop, Printer, and Tablet were the top-selling products.
